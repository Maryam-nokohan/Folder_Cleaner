# Folder Cleaner (Go)

A simple command-line tool written in Go that organizes files in a directory by moving files with a specific extension into a chosen folder.

# Features

- Read a directory path from the user
- Filter files by extension (e.g. cpp, mp3)
- Create a destination folder automatically
- Move matching files into that folder
- Uses only Go standard library

# How to use 

```shell
git clone  https://github.com/Maryam-nokohan/Folder_Cleaner.git
go run main.go
```
